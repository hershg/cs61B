Lecture 03 - 01/23

Primitive Types

  • Computer stores info as bits in memory
  • Java has 8 primitive types to interpret bits:
    • byte, short, int, long, float, double, boolean, char
  • When declaring a variable of certain type, computer sets aside appropriate memory size of empty bits to store variable value
  • When initializing a declared variable, bits set to appropriate 0's and 1's
  • Java prevents us from accessing uninitialized variable
  • for some int x, y = x copies bits from x into y. Thus changing y doesn't affect x

Reference Types

  • Everything that isn't among those 8 primitive types is a 9th type: a reference to an object
  • When instantiating an object, Java allocates memory for each instance variable of the class, and gives a default value (ex: 0, null)
  • Constructor then can fill these boxes with value of passed parameters
  • We can think of new as returning the memory address of the newly created object
    • addresses in java are 64 bits
  • So, when we say myObject x = new myObject();, we are really setting x as a pointer to the address of the object. Thus, x is a 64 bit address value

  • Here, when we do y = x, y copies the bits from x so y is also now a pointer to the same object. Thus changing y also changes x

  • This gives us the Golden Rule of Equals:

    Given variables x and y, y = x copies bits from x into y

Parameter Passing

  • Parameter Passing obeys same rule: simply copies bits into new scope (a.k.a. pass by value)
  • If a pointer variable is made to point to a new object, it no longer has an impact on the object it initially pointed to. For example:
    myObject x = new myObject(param0);
    y = x;
    y = new myObject(diffParam0);
    Here, y originally points to what x points to, but since after that y points to a new object, changing y will not affect x

Declaring, Instantiating, and Assigining Arrays

  • Arrays are objects
  • Can declare with int[] myArray;
  • Can instantiate with myArray = new int[2];
  • Can assign with myArray[0] = 5;myArray[1] = 3;
  • Can do all at same time with int[] myArray = new int[]{5, 3};
